◆ cli_app

typedef struct cli_app_t cli_app

Represents the top-level CLI APP object.

The cli app contains all the commands available, and their associated handlers.

By default it contains a command with the name :CLI_ROOT_COMMAND_NAME. The root command is activated with the options and arguments, if no other command name is matched.
